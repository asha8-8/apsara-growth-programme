Interactive walkthrough presentation for the **Greater Bhubaneswar Growth Partner Programme** by Apsara Realcon Pvt. Ltd.

## About

A single-file, zero-dependency, offline-capable interactive HTML presentation covering:

- Programme structure and investment overview
- Financial returns and cumulative cash outcomes
- Dual exit paths (Sale Deed or Buyback)
- Location advantages and proposed amenities
- Legal framework and investor safeguards
- Comparative analysis vs. alternative instruments
